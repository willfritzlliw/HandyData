# HandyData
 A collection of prefilled python data structures saved for quick reuse.

## Files
- `lastnames.py`: A list of last names.
- `states.py`: A list of US states, abbreviations, and a dictionary mapping abbreviations to states.
- `words.py`: A collection of lists of words, including verbs, adjectives, nouns, pronouns, colors, places, jobs, animals, and conjunctions.
- `years.py`: A list of years from 1950 to 2021, and a dictionary mapping the last two digits of a year to the full year.
